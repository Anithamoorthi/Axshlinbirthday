# 🎂 Axyyy's Birthday Surprise Website

A premium, interactive, and emotional multi-page birthday surprise website built with **React**, **Vite**, and **React Router v6**.

---

## 🎵 Media Setup Instructions

Before running the website, replace the placeholder files in the `src/assets/` directory with your actual high-quality media files. The files should match these exact names:

### Photos (in `src/assets/`)
- `photo1.jpg` — Background image for **Page 2** (Photo Page)
- `photo2.jpg` — Background image for **Page 3** (Letter Page)
- `photo5.jpg` — Background image for **Page 5** (Final Page)

### Videos (in `src/assets/`)
- `video1.mp4` — The first video played on **Page 4** (Video Page)
- `video2.mp4` — The second video played sequentially on **Page 4**

### Audio / Songs (in `src/assets/`)
- `thangapoovey.mp3` — Song that plays automatically on **Pages 2 and 3**
- `senthamaraiye.mp3` — Song that loops automatically on **Page 5** (Final Page)

---

## 🚀 How to Run the Project

Follow these steps to run the application locally:

1. **Install Dependencies** (if not already done):
   ```bash
   npm install
   ```

2. **Start the Development Server**:
   ```bash
   npm run dev
   ```

3. **Open in Browser**:
   Open the URL shown in the terminal (usually `http://localhost:5173`).
